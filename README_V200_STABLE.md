# Tasneef V200 Stable

هذه نسخة محافظة من النظام، الهدف منها تثبيت التشغيل الحالي بدون تغيير آلية العمل.

## ملفات التشغيل الأساسية
- `index.html` صفحة الدخول
- `admin.html` صفحة الإدارة
- `supervisor.html` صفحة المشرف
- `technician.html` صفحة الفني
- `client-report.html` تقرير العميل
- `app.js` ملف التشغيل الرئيسي
- `style.css` ملف التصميم

## قبل الرفع
ارفع محتويات هذا المجلد كما هي على الاستضافة.
لا تحذف مجلد `sounds` ولا ملفات الصور.
