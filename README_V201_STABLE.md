# Tasneef V201 Stable

هذه نسخة محافظة مبنية على V200، ولا تغير آلية العمل الأساسية.

## أهم التعديلات
1. إصلاح حسابات المخزون حتى لا يتضاعف رقم خرج عند وجود أمر صرف وحركة مخزون لنفس العملية.
2. توضيح ملخص تفاصيل المنتج:
   - الصادر لأوامر الصرف
   - المرتجع
   - صافي الصادر
   - الاستهلاك المباشر
   - إجمالي المستهلك
   - المتبقي
3. تحسين طباعة التقارير بحيث تعتمد على الجدول الحالي بعد التحديث والفلترة، مع تنظيف كاش التقارير بعد الحذف.
4. إضافة مصفوفة صلاحيات لكل تبويب داخل إدارة المستخدمين:
   - مشاهدة
   - إضافة
   - تعديل
   - حذف
   - طباعة/تصدير
