تقرير إصلاح مشاريع نسخة المشرف — V10813
========================================

المشكلة
-------
نسخة المشرف كانت تعرض المشاريع التي يكون فيها projects.supervisor_id مطابقًا للمشرف فقط.
بعض المشاريع موجودة في النظام الموحد والتوزيع الشهري، لكن معرف المشرف داخل جدول projects قديم أو غير محدث؛ لذلك كانت تختفي من قائمة المشرف.

الإصلاح
-------
تمت إضافة وحدة واحدة:
  tasneef_supervisor_projects_complete_v10813.js

الوحدة تجمع مشاريع المشرف من:
1. الربط المباشر داخل projects.
2. جدول monthly_distribution للشهر الحالي.
3. جدول workers وربط العامل بالمشرف.
4. جدول worker_project_assignments.

عند عدم وجود توزيع مطابق في الشهر الحالي فقط، تستخدم بيانات الشهر السابق كحل احتياطي.

طريقة المطابقة
-------------
- معرف حساب المشرف.
- كود الموظف أو كود المشرف.
- اسم المشرف بعد توحيد الحروف العربية.

النتيجة
-------
- ظهور جميع المشاريع المخصصة للمشرف دون تكرار.
- تحديث قوائم المشاريع في:
  * الدخول والخروج.
  * الحضور والغياب.
  * التكتات.
  * الأوردرات.
  * طلبات المخزن.
  * التقرير اليومي للعميل.
- منع فلتر النطاق القديم من حذف المشاريع الصحيحة التي مصدرها التوزيع الشهري.
- عدم تعديل أي بيانات في Supabase؛ التوحيد يتم داخل ذاكرة صفحة المشرف فقط.
- لا يحتاج تشغيل SQL.

---------
